# Quiz: The System Prompt

> **Live lesson:** https://langchain-ai.github.io/lca-deepagents/m1/m1.4-system-prompt-quiz.html

**Q1: What does the `system_prompt` argument let you provide?**

A) Custom instructions for the agent
B) A replacement for the model
C) A new tool schema
D) A LangSmith project name

*Answer: A. `system_prompt` is where you provide custom instructions such as persona, domain, or constraints.*

---

**Q2: What does `create_deep_agent` do with your `system_prompt`?**

A) It combines it with Deep Agents' built-in base instructions
B) It sends it as a user message
C) It turns it into a tool
D) It stores it only in LangSmith

*Answer: A. Your instructions are combined with the built-in base instructions to form the full system prompt sent to the model.*

---

**Q3: You write `SYSTEM_PROMPT = 'You are a pirate.'` but the agent replies with no pirate tone. What is most likely wrong?**

A) The built-in instructions override all personas
B) The model requires a special flag to enable custom personas
C) SYSTEM_PROMPT was defined but not passed to `create_deep_agent`
D) Persona instructions only work with tools

*Answer: C. Defining the string is not enough. It must be passed explicitly: `create_deep_agent(model=model, system_prompt=SYSTEM_PROMPT)`.*

---

**Q4: What are the built-in base instructions?**

A) The `system_prompt` string you pass to `create_deep_agent`
B) The tool schemas appended by the SDK before each LLM call
C) Deep Agents' default behavior guidance that is always present
D) The user's first chat message

*Answer: C. The built-in base instructions are Deep Agents' default behavior guidance. They remain present whether or not you pass `system_prompt`.*

---

**Q5: You switch from a pirate persona to a cowboy persona by changing `SYSTEM_PROMPT`. What else changes?**

A) The built-in base instructions adapt to match the new persona
B) Only your custom instruction changes
C) The model changes automatically
D) The tool schemas update to reflect the new persona's capabilities

*Answer: B. Only your custom instruction changes. The built-in base instructions stay put.*
