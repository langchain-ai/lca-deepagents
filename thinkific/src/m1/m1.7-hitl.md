You used Human-in-the-Loop (HITL) in an earlier lesson. In this lesson, you'll learn why it's essential for production agentic systems.

A deployed agent consumes computing resources while it is running. At scale, thousands or millions of deployed agents can consume significant resources.

Without Human-in-the-Loop, an agent waiting for user input may continue consuming compute resources unnecessarily. With Human-in-the-Loop, the agent can suspend execution, save its state, and release those resources. When the user responds, the agent resumes from its saved state instead of remaining active while it waits.
