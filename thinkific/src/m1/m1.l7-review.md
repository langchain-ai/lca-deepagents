# Module 1 Review

## Recap

- A Deep Agent = a **model** + a **system prompt** the SDK assembles every call.
- **Swapping models** is a one-line change in `models.py`; the assembled prompt structure around
  the model is identical no matter the provider.
- `create_deep_agent` assembles three parts, **USER → (BASE or CUSTOM) →
  SUFFIX**; then **mandatory middleware append** the `write_todos`, filesystem,
  and `task` sections after them.
- **BASE** is the SDK's built-in *behavior* prompt (`BASE_AGENT_PROMPT`), not the
  provider's; the **filesystem** instructions and tools come from the mandatory
  `FilesystemMiddleware`, not from BASE.
- **USER** is your `system_prompt`; it sits at the front of the assembled prompt.
- **SUFFIX** is harness-profile tuning; our `claude-haiku-4-5` default ships one.
- You can read the whole assembled `system` message for any run in **LangSmith**.
