# Creating a Deep Agent

## Using a Coding Agent

Before we explore, let's get a bare agent running. Paste the prompt below into
your coding agent (Claude Code, Cursor, etc.) and let it build the file.

<Video src="https://share.descript.com/view/k7KXh34BOdZ" />

```text
In lca-deepagents/python/m1/scratch_agent.py, create a minimal deep agent:
import the model from python/models.py, call create_deep_agent(model=model)
with no other arguments, invoke it with "What is an LLM?" and print the last
message's content.
```

<details>
<summary>No coding agent? Create the file manually [click to expand]</summary>
<br>

Create `python/m1/scratch_agent.py`:

```python {5,8,9,10}
import sys
from pathlib import Path
sys.path.insert(0, str(Path(__file__).resolve().parent.parent))
from models import model

from deepagents import create_deep_agent

agent = create_deep_agent(model=model)
result = agent.invoke({"messages": [{"role": "user", "content": "What is an LLM?"}]})
print(result["messages"][-1].content)
```

Then run it:

```bash
cd ~/Documents/Github/lca-deepagents/python && uv run ./m1/scratch_agent.py
```

</details>

Once your coding agent has created the file, run it:

```bash
cd ~/Documents/Github/lca-deepagents/python && uv run ./m1/scratch_agent.py
```
