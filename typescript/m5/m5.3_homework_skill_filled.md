# Top Artists Report

A metrics task, the same shape as territory-report but sliced by artist
instead of genre.

## 1. Gather the metrics

Ask **chinook-analyst** for Jane's book of business (`SupportRepId = 3`):

- Revenue by artist, across all of Jane's invoices.
- The top 5 artists by revenue, with exact amounts.

## 2. Chart

Call `render_pie_chart` with the top 5 artists' names and revenue amounts,
saved as `top_artists_chart.png`.

## 3. Write the report

- Use the code interpreter to get today's date.
- `write_file` a short Markdown report to `/outputs/top_artists-<date>.md`:
  the top 5 artists with their revenue, and a line noting the chart.

## Done

Tell Jane where the report and chart were saved, and name the single
top-revenue artist.
